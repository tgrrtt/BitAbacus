Info

Portfolio, help, and contact information.

What is BitAbacus?

BitAbacus is a web service that keeps track of your cryptocurrency holdings. As more and more cryptocurrencies are introduced, and as the price fluctuates wildly, it can be hard to tell exactly what your cryptocurrency net worth is at any given moment. BitAbacus aims to take all the work out of this by automatically retrieving your cryptocurrency info from exchanges and wallets and showing you current conversions to bitcoin and fiat.

How does it work?

BitAbacus uses info APIs to get information on your holdings on various exchanges. This means your info will be kept current.

How secure is it?

Everything is secure as long as trade or withdrawal APIs are not used. Supported exchanges allow the user to set the permissions of the API. As long as the APIs in use on BitAbacus are set to "info only" on their respective exchanges, no harm can be done.

How do I set up the APIs?

Log into the exchange you with to pull data from. Go to the API section, and create a new API for use on this site. Either before creaating it or afterwards, you will have the option to select the permissions of the API. Only select "info". Copy and paste your API key and secret onto their respective fields under the portfolio settings page of this site.
